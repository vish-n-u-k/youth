Requirements (8)

PP-REQ-01unknown

Display program name, age group, date range, enrollment count vs capacity, and status (draft, active, closed)

Acceptance Criteria

All fields visible for each program; list is sortable by name, start date, and status

PP-REQ-02unknown

Admin can set: program name, age group, number of sessions, session duration, location, start date, end date

Acceptance Criteria

All fields are required for publishing; draft programs can have incomplete fields

PP-REQ-03unknown

Program has status: draft, active, closed

Acceptance Criteria

Only 'active' programs are visible to parents; admin can change status at any time

PP-REQ-04unknown

Admin can set a base program fee (total or per-session rate × number of sessions)

Acceptance Criteria

Fee can be entered as total amount or calculated from per-session rate; final amount displays correctly

PP-REQ-05unknown

Admin can set a registration fee and toggle it as required or waived at the program level

Acceptance Criteria

When set to 'waived', registration fee is not included in enrollment price breakdown; when 'required', it is added

PP-REQ-06unknown

Admin can apply per-enrollment manual overrides: waive registration fee, give free sessions

Acceptance Criteria

Override is applied to a specific enrollment and reflected in that enrollment's price breakdown without affecting the program-level pricing

PP-REQ-07unknown

Admin can create coupons with: code, discount type (percentage or flat amount), applicable scope (program fee only, registration fee only, or both), optional expiry date, optional usage limit

Acceptance Criteria

Coupon with all fields can be created, edited, and deactivated; deactivated coupons are not accepted during enrollment

PP-REQ-08unknown

Parent can enter a coupon code during enrollment and see the discount applied to the price breakdown in real time

Acceptance Criteria

Valid coupon reduces the displayed total; invalid or expired coupon shows an error message; discount is applied to the correct fee component based on scope
